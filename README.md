# 🌤 Weather Application (JavaScript Project)

## 📌 Description

This project is a simple **Weather Application** built using **HTML, CSS, and JavaScript**.
It allows users to enter a city name and fetch real-time weather details using the **OpenWeatherMap API**.

The application displays temperature, weather condition, and humidity in a clean and user-friendly interface.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡 Displays temperature in Celsius
* 🌥 Shows weather description
* 💧 Displays humidity
* ⚠ Handles errors (invalid city name)
* 🎨 Attractive UI with gradient background

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Fetch API)
* OpenWeatherMap API

## 🧠 How It Works

* User enters a city name
* JavaScript sends a request using **Fetch API**
* API returns weather data in JSON format
* Data is displayed dynamically on the page
